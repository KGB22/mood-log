🌿 Mood Log

## 📚 프로젝트 소개

간단히 감정 일기를 작성하고 감정 데이터를 차트로 확인할 수 있는 프로젝트입니다.

“마음을 돌보는 습관”을 목표로 간단한 일기 작성 → 연간/월간 차트 시각화로 최종 내 감정 데이터 확인 루틴을 제공합니다.

## 🔎 서비스 주요 기능

✅ **일기 작성**

- 날짜, 감정 선택, 일기 내용 작성 가능

✅ **월별 감정 차트**

- 각 달별 가장 많이 느낀 감정 확인
- 월별 상태 흐름 확인 가능

✅ **연간 감정 차트**

- 올해 가장 많이 느낀 감정 TOP 3 시각화

✅ **로컬스토리지 기반 영속성**

- 새로고침해도 데이터 유지

## 🛠️ 주요 기술 스택

### ⚛️ Frontend

- React.js
- TypeScript
- React Router
- Styled-Components
- Recharts

## 📊 데이터

- **로컬스토리지 기반 데이터 관리**
- 감정 ID, 이름, 이미지, 색상, 작성 일자, 내용으로 구성
